# Fishing Game Prototype (RBXL)

## How to Use

1. Download the `.rbxl` file from this repository  
2. Open it in Roblox Studio  
3. Press **Play** to run the game  

## Basic Controls

- Equip the fishing rod  
- Click to cast  
- Wait for a bite (rod will highlight)  
- Click again to start the minigame  
- Hit zones to catch fish  

## Multiplayer

- When a player has a fish on the line, others nearby can help  
- Stand near them and join when prompted  
- Helping makes catching easier and rewards everyone  

## Inventory

- Press **C** to open the inventory  
- Shows all fish types  
- Uncaught fish appear as silhouettes  
- Click a fish to view your catches  

## Debug

- **Shift + Click** a fish in the inventory to instantly start its minigame
